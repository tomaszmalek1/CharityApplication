Celem projektu jest stworzenie miejsca, w którym każdy będzie mógł oddać niepotrzebne rzeczy zaufanym instytucjom.

Użyte technologie
- Spring Boot
- Spring Form
- Spring Security
- Hibernate
- Spring Data JPA
